# Homomorphic Encryption + Federated Learning (HE+FL)

This repo contains a privacy-preserving CNN training pipeline combining Federated Learning and Homomorphic Encryption.

- macOS dev uses a Pyfhel shim for local runs.
- Docker/Linux provides real Pyfhel execution.

Key paths:
- repo_HE_FL/: notebook + code
- data/: datasets (MedMNIST exports)
- weights/: model weights and encrypted blobs
